  1. はじめに
================================================================================

Counter Counter をダウンロードいただき、ありがとうございます！

このアプリケーションは、配信中にカウンターを簡単に操作・表示できるように
設計されたツールです。グローバルホットキーでゲーム中でも操作可能です。

================================================================================
  2. 動作環境
================================================================================

【必須】
- OS: Windows 10 (64bit) 以降 / Windows 11 (64bit)
- メモリ: 4GB以上
- ディスク空き容量: 100MB以上
- .NET 8 Runtime（本アプリに同梱）

【推奨】
- OBS Studio 最新版
- インターネット接続（初回起動時のみ）

================================================================================
  3. インストール方法
================================================================================

【方法1: ZIP版（推奨）】
1. ダウンロードしたZIPファイルを解凍
2. 適当なフォルダに配置（例: C:\Apps\CounterCounter\）
3. CounterCounter.exe を実行

【方法2: インストーラ版】
1. CounterCounter_Setup.exe を実行
2. インストールウィザードの指示に従う
3. デスクトップまたはスタートメニューから起動

【重要】
- 初回起動時、Windows Defender SmartScreen の警告が出る場合があります
- 「詳細情報」→「実行」をクリックしてください
- 本アプリにウイルスは含まれておりません

================================================================================
  4. 使い方
================================================================================

--------------------------------------------------------------------------------
  4.1. 初回起動
--------------------------------------------------------------------------------

1. CounterCounter.exe をダブルクリック
2. タスクトレイにアイコンが表示されます
3. タスクトレイアイコンを右クリック → 「設定を開く」

--------------------------------------------------------------------------------
  4.2. サーバー起動
--------------------------------------------------------------------------------

1. 設定画面の左側メニュー「サーバー設定」をクリック
2. 「サーバー起動」ボタンをクリック
3. 緑色の「起動中」表示になれば成功です

【または】
- タスクトレイアイコンを右クリック → 「サーバー起動」

--------------------------------------------------------------------------------
  4.3. OBSへの設定
--------------------------------------------------------------------------------

【スライドイン表示（変更時のみ表示）】
1. OBSで「ソース」→「ブラウザ」を追加
2. URLに以下を入力:
   http://localhost:9000/obs.html
3. 幅: 800px、高さ: 600px を推奨
4. 「完了」をクリック

【ローテーション表示（常時表示・自動切り替え）】
1. OBSで「ソース」→「ブラウザ」を追加
2. URLに以下を入力:
   http://localhost:9000/rotation.html
3. 幅: 800px、高さ: 600px を推奨
4. 「完了」をクリック

【ヒント】
- 設定画面の「接続情報」からURLをコピーできます
- ポート番号が変更されている場合は、表示されているURLを使用してください

--------------------------------------------------------------------------------
  4.4. カウンター操作
--------------------------------------------------------------------------------

【方法1: グローバルホットキー】
- デフォルト設定:
  - 増加: Ctrl + Shift + ↑
  - 減少: Ctrl + Shift + ↓
  - リセット: Ctrl + Shift + R
  - ローテーション切り替え: Ctrl + Shift + N

- ゲーム中でも動作します！

【方法2: 設定画面から】
- 設定画面の「カウンター管理」で「+」「-」「リセット」ボタンをクリック

【方法3: タスクトレイから】
- タスクトレイアイコンを右クリック → 「設定を開く」

================================================================================
  5. 主な機能
================================================================================

【カウンター管理】
- 複数のカウンターを作成・管理
- カウンター名・色をカスタマイズ
- カウンター値の増減・リセット

【グローバルホットキー】
- ゲーム中でも操作可能
- カウンターごとに最大3セットのキー設定
- 衝突検知機能

【OBS連携】
- スライドイン表示（変更時のみ）
- ローテーション表示（常時・自動切り替え）
- リアルタイム更新（WebSocket）

【アニメーション】
- フラッシュエフェクト
- スライドインアニメーション
- ローテーション演出

【その他】
- タスクトレイ常駐
- 設定の自動保存
- ポート自動選択

================================================================================
  6. トラブルシューティング
================================================================================

【Q1. サーバーが起動しない】
A1. ポート 9000 が使用中の可能性があります。
    「サーバー設定」でポート番号を変更してください。

【Q2. OBSで表示されない】
A2. 以下を確認してください:
    - サーバーが起動しているか
    - ブラウザソースのURLが正しいか
    - ファイアウォールでブロックされていないか

【Q3. ホットキーが反応しない】
A3. 以下を確認してください:
    - サーバーが起動しているか
    - 他のアプリケーションとキーが衝突していないか
    - カウンター編集でホットキーが正しく設定されているか

【Q4. WebSocketに接続できない】
A4. サーバーを再起動してください。
    タスクトレイ → 「サーバー停止」→ 「サーバー起動」

【Q5. 設定が保存されない】
A5. config.json ファイルが読み取り専用になっていないか確認してください。
    また、管理者権限で実行してみてください。

【Q6. アプリが起動しない】
A6. .NET 8 Runtime がインストールされているか確認してください。
    本アプリには同梱されていますが、古いバージョンをお使いの場合は
    Microsoft公式サイトから最新版をダウンロードしてください。

【それでも解決しない場合】
- config.json を削除して再起動
- アプリケーションを再インストール

================================================================================
  7. アンインストール
================================================================================

【ZIP版】
1. CounterCounter.exe のフォルダごと削除
2. タスクトレイからアプリを終了してから削除してください

【インストーラ版】
1. Windowsの「設定」→「アプリ」→「インストールされているアプリ」
2. 「Counter Counter」を選択 → 「アンインストール」

【設定ファイルの削除】
- config.json（アプリと同じフォルダ）を削除
